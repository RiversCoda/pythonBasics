# 年月日控制
## 输入一个正实数分别输出整数和小数部分
```py
y,m,d=eval(input())
print("我的出生日期是{0}年{1:02}月{2:02}日".format(y,m,d))
```
## 输入
## 1999,5,1
## 输出
## 我的出生日期是1999年05月01日


# 输入一个正实数分别输出整数和小数部分
```py
x = float(input("请输入一个正实数:"))
k = int(x)
f = x - k
print("整数部分是{0},小数部分是{1}".format(k,f))
```
## 输入
## 3.14
## 输出
## 整数部分是3,小数部分是0.14000000000000012
